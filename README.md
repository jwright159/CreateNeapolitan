# Create: Neapolitan

Compatibility between [Create](https://modrinth.com/mod/create) and [Neapolitan](https://modrinth.com/mod/neapolitan).

Also supports [Create Crafts & Additions](https://modrinth.com/mod/createaddition)'s cakes!

Download on [Modrinth](https://modrinth.com/mod/create-neapolitan) or [Curseforge](https://www.curseforge.com/minecraft/mc-mods/create-neapolitan) for Minecraft 1.21.1 Neoforge.

## Features

- Fluid versions of all 6 ice creams.
- New "plain" ice cream crafting base that acts as a milk bottle.
- Chocolate spouting recipes for Neapolitan's chocolate-covered foods.
- Snow crafting loop, for if you don't want to use snow golems.
- Mixing recipes for Neapolitan foods using liquid milk.
- Neapolitan sequenced assembly recipe.
